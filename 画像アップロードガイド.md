# 画像アップロードガイド

## 必要な画像ファイル

以下の2つの画像を `images/` フォルダに配置してください：

### 1. 代表写真
- **ファイル名**: `representative.jpg`
- **配置場所**: `images/representative.jpg`
- **説明**: 福原大貴代表の個人写真
- **推奨サイズ**: 幅300px以上、正方形または縦長の比率

### 2. メンバー集合写真
- **ファイル名**: `team.jpg`
- **配置場所**: `images/team.jpg`
- **説明**: チームメンバーの集合写真（13名のグループ写真）
- **推奨サイズ**: 幅900px以上、横長の比率

## 画像のアップロード方法

### 方法1: ファイルエクスプローラー/ Finderを使用

1. `images/` フォルダを開く
2. 画像ファイルをドラッグ&ドロップまたはコピー&ペースト
3. ファイル名を上記の名前に変更

### 方法2: ターミナルを使用

```bash
# imagesフォルダに移動
cd images

# 画像ファイルをコピー（実際のファイルパスに置き換えてください）
cp /path/to/your/representative-photo.jpg representative.jpg
cp /path/to/your/team-photo.jpg team.jpg
```

## 画像形式について

- **推奨形式**: JPG, PNG, WebP
- **ファイルサイズ**: 1MB以下を推奨（ページの読み込み速度のため）
- **最適化**: 必要に応じて画像圧縮ツールを使用してください

## 画像が表示されない場合

画像ファイルが正しく配置されていない場合、プレースホルダーが表示されます。以下の点を確認してください：

1. ファイル名が正確か（大文字小文字を含む）
2. ファイルが `images/` フォルダ内にあるか
3. ファイルの拡張子が正しいか（.jpg, .jpeg, .png など）

## 画像のプレビュー

ブラウザで `index.html` を開いて、画像が正しく表示されているか確認してください。
